## 🚀 Quick Start

### Backend

```bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
# source venv/bin/activate  # Linux/Mac
pip install -r requirements.txt
cp .env.example .env
# Edit .env with your API keys
python seed_data.py
python app.py
```

### Frontend

```bash
cd frontend
npm install
cp .env.example .env
# Edit .env with your backend URL
npm run dev
```

## 🔑 Environment Variables

### Backend (.env)
```

VITE_API_URL=http://localhost:5000
```

## 📦 6 Core Innovations

### 1. 🎭 Mood-Based AI Recommendations
Select from Adventurous, Romantic, Budget, or Comfort moods. The AI dynamically adjusts recommendations based on your emotional context and current season.

### 2. 🎤 Voice Search
Speak naturally: "Find a cozy Italian place under $20 for a date night." Web Speech API converts your voice to an intelligent query.

### 3. 📊 Real-Time Social Proof
Live badges: "12 people viewed this in the last hour" — powered by real-time event tracking in Supabase.

### 4. 🎁 AI Bundle Builder
Smart bundles like "Date Night: Dinner + Dessert + Movie" generated by Gemini AI based on your mood and preferences.

### 5. 🧠 Explainable AI
Every recommendation card shows WHY: "Recommended because you love Italian food and this place has a 4.8★ romantic ambiance score."

### 6. 📈 Business Owner Dashboard
Professional analytics: views, clicks, purchases, activity timeline — all visualized with Recharts.

## 🌦️ Seasonal Intelligence

The system automatically detects the current season and adjusts recommendations:
- **Spring** 🌸 — Outdoor dining, fresh seasonal menus
- **Summer** ☀️ — Cool treats, rooftop bars, water activities
- **Autumn** 🍂 — Cozy cafes, harvest menus, warm drinks
- **Winter** ❄️ — Comfort food, indoor experiences, holiday specials
